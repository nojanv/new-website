Predicting Soccer Match Outcomes

CS4641@gatech

machine learningdevelopment

Soccer matches, and matches of all professional sports, are intrinsically difficult to predict. As Felipe, a colleage of who helped create this project, put it, "If it were easy to predict games, then there'd be no point in playing them." Nevertheless, using different classifiers, we built several models using Python, Jupyter Notebook, and machine learning / data science libraries (numpy, matplotlib, scikit-learn etc.) to predict the outcomes of World Cup matches. Our attemps reached ~75% accuracy in predicting games. Given this was our first foray into machine learning, we consider it great success.
